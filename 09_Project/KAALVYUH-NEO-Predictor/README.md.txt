## 📖 Overview

KAALVYUH is a **Machine Learning-based Near-Earth Object (NEO) Hazard Prediction System** developed using NASA's Near-Earth Object dataset.

The project predicts whether an asteroid is **Hazardous** or **Non-Hazardous** using a trained **Random Forest Classifier**. An interactive **Streamlit dashboard** allows users to enter asteroid parameters and receive instant predictions along with visual insights.

The objective is to demonstrate how Machine Learning can automate asteroid hazard classification and support faster, data-driven decision making for planetary defense and scientific research.

---

# 🌌 Problem Statement

NASA monitors thousands of Near-Earth Objects every year to identify asteroids that may pose a potential threat to Earth.

Traditional analysis of large orbital datasets is time-consuming and requires significant manual effort.

KAALVYUH addresses this challenge by using Machine Learning to automatically classify Near-Earth Objects as Hazardous or Non-Hazardous, enabling faster and more reliable predictions.

# 📌 Input Parameters

The prediction model accepts the following asteroid characteristics:

- Estimated Diameter
- Relative Velocity
- Miss Distance
- Absolute Magnitude

These parameters are processed by the trained Random Forest model to predict whether the asteroid is hazardous.
